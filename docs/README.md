# Documentation

This folder contains lightweight documentation for maintaining the site.

## Topics
- How to add or update pages
- How to keep English and French content in sync
- Where shared assets and scripts live

## Recommended workflow
1. Create or update the relevant page in the appropriate language folder.
2. Keep links and asset paths relative to the page location.
3. Preview the change locally before publishing.
4. If a new reusable pattern appears, capture it in the templates folder.
